## GroceryManager

GroceryManager is an Android app (Kotlin + Jetpack Compose) to manage kitchen storage, recipes, and shopping flow in one place.

### Features
- Storage tracking with quantity-aware and presence-only ingredients
- Recipe management with cookable/missing ingredient logic
- Shopping list generation from recipe gaps
- Move bought items into storage
- Ingredient/category admin section (edit/delete)
- Persistent local data with Room
- Dark mode toggle

### Tech Stack
- Kotlin
- Jetpack Compose (Material 3)
- Room (SQLite)
